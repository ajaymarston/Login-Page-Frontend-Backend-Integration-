# React Login & Sign Up Authentication UI

A simple authentication interface built using **React**, **React Router**, and **Tailwind CSS**.  
The project demonstrates a basic **Login and Sign-Up workflow**, allowing users to register with an email and password and then authenticate using those credentials.

This project focuses on **React state management, routing, and UI styling**.

---

## Features

- User **Sign-Up page** for registering email and password  
- **Login page** for authentication  
- **Password visibility toggle**  
- **Navigation between Login and Sign-Up using React Router**  
- **Background image UI design**  
- **Tailwind CSS styling**  
- Simple **client-side authentication logic using React state**

---

## Tech Stack

- React.js  
- React Router  
- JavaScript (ES6)  
- Tailwind CSS  
- HTML5
